Q: How do I start?

If you have been chosen for the Convention Tycoon pre-pre-pre-alpha, you have received game media via flash drive or download link. Run "ConventionTycoon.exe" to bring up a launcher that will allow you to change some settings and start the game.

Top

Q: How do I change the game controls?

A: When you first launch the game, you should see "Graphics" and "Input" tabs. Click on "Input" and double-click on the action you would like to change the controls for. Press the key you want to use for that action, and your changes will be applied.
